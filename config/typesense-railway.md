## Environment Variables

**Setup:** Get the Typesense API key from Railway dashboard → Typesense service → Variables.

```env
TYPESENSE_API_KEY=<admin-key-from-railway-dashboard>
```

**Note:** This key is auto-generated by the Railway template. Store securely in your password manager.

## Search Performance

**Initial benchmarks (2026-01-23):**

| Query | Hits | Time |
|-------|------|------|
| 'prorail' | 2 | 13ms |
| 'amsterdam' | 2,916 | 18ms |
| 'subsidie' | 502 | 15ms |
| 'ns' | 60 | 21ms |

**Target:** <100ms | **Achieved:** <25ms ✅

**Production Performance (2026-02-09) with 463,731 documents:**
- Warm queries: ~100-150ms (includes Typesense + database lookups)
- Cold queries: ~200ms (was ~200ms regex-only fallback before Typesense)
- Autocomplete: <50ms (Typesense-only, no database)
